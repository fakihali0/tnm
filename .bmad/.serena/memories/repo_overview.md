# Repository Overview
- Two main areas: `tnm_concept/` contains the Lovable-generated Vite + React + TypeScript web app, while `docs/` houses PRD, epics, sprint tracking, and story outputs.
- App integrates Supabase (env vars `VITE_SUPABASE_URL`, `VITE_SUPABASE_ANON_KEY`), shadcn-ui, Tailwind CSS, Framer Motion, zustand, and various Radix UI primitives.
- Documentation folder includes MT5 integration requirements, completion summaries, and live sprint + story assets consumed by Scrum workflows.
- Work happens on macOS (Darwin). Node.js/npm workflow via Vite; stories/deliverables saved under `docs/stories`.
