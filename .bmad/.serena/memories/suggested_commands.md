# Suggested Commands
- `cd tnm_concept && npm install` — install dependencies.
- `cd tnm_concept && npm run dev` — start Vite dev server.
- `cd tnm_concept && npm run build` — production build.
- `cd tnm_concept && npm run preview` — preview built app locally.
- `cd tnm_concept && npm run lint` — run ESLint across the project.
- `cd tnm_concept && npm run test` — run Vitest suite.
- Story workflows: run BMAD commands from repo root (e.g., `*create-story`) saving outputs to `docs/stories`.
